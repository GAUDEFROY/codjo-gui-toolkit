Ce package contient des composants JAVA BEANS.

Description des JAVA BEANS
  1. SyntaxControlPanel
      Panel permettant de vérifier la syntaxe d'une expression DynamicJava située dans un JTextArea.
      Il vérifie la cohérence :
        - des parenthèses (autant d'ouvrantes que de fermantes)
        - des opérateurs logiques ("&&" et "||")
        - des opérateurs de comparaison ("==" et "!=")
        - des doubles quotes (nombre pair)
      Pour cela, il suffit de placer ce panel dans la fenêtre où se trouve le JTextArea de l'expression et de lui positionner ce dernier par la méthode "setExpressionTextArea".
