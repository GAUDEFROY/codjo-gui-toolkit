Composants graphiques permettant d'afficher un JFileChooser avec de nombreux accessoires (Recherche, Favoris, ...)

Utilisation du package

Pour afficher le JFileChooser utilisé la classe FileChooserManager comme factory. 

FileChooserManager.showChooserForOpen("c:/toto.txt", "Choisir un fichier");
